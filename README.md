## ‚ú® Quick Overview
This project builds an **end-to-end ML pipeline** that predicts the presence of **heart disease (0/1)** using clinical patient attributes, combining:
- **EDA + clinical interpretation**
- **Statistical testing (t-test, chi-square)**
- **ML models (LR, DT, RF, SVM) + evaluation (ROC/AUC, F1, etc.)**
